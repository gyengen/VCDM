Output

VCDM model output is shown as three sub-plots with each having the same time period x-axis (seconds). There is an option to download the results as an MS Excel file. Note that this model version assumes the initial pipe condition to have unrealistic fully developed material layers. As a result initial turbidity responses are invalid until material layers achieve equilibrium with background flows.
  • PLOT 1: The value of the input flow translated to applied shear stress in N/m2 (Pascal),
  • PLOT 2: Three example tracked shear strength bands (determined as 25th, 50th and 75th percentiles of the current hydraulic conditions) to represent a cross-section of the shear stress range as defined by the input flow range. The y-axis is amount of material present (layer condition) corresponding to specific shear strength band where 0 = no material and 1 = a fully developed material layer.
  • PLOT 3: Simulated (and if part of input file the measured) turbidity in NTU.
