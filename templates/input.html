Input Data Format

The amount of wall bound discolouration material within discrete shear strength bands and output simulations described by the VCDM is defined by imposed hydraulics. A time series flow profile therefore needs to be submitted. The option to include time series turbidity data to aid calibration is also part of the input process. With this online version the following format is required;

  • File submission must be .csv (comma-separated values) format
  • Input file must have the following headers and data in the correct units as demonstrated below;
Time (s)Flow (m3/s) Time (s)NTU
00.105960000.15
9000.107463000.18
18000.108866000.21

  • File format is critical to prevent errors;
    1. Times must be sequential with first flow value at time = 0. Time steps do not need to be equal.
    2. Turbidity data (columns 3 & 4) may be left blank. When used, turbidity time series is independent of flow, so different time periods may be used and there is no restriction on initial time stamp/value.
    3. Flows values must be non-zero and positive with no gaps or non-numeric terms.
    4. The number of turbidity data points must not exceed flow data (i.e. columns 3 & 4 must be no larger than 1 & 2). Note excessive decimal places (>6) may cause computational issues.

Select a file to upload

The selected file must be in .csv (comma-separated values) format. The size of the input file must be smaller than 32Mb.
